# Terminal-Apps-for-Linux
Это терминальные программы, не использующие GUI для ос Linux

# Калькулятор
Калькулятор представлен в calculator.py, работает на библиотеках curses и math, при желании можно обойтись и без них. После запуска занимает около 4,5 мб ОЗУ в отличие от gnome calculator с 22 мб ОЗУ.
